Overview

_.py is a collection of libraries to rapidly assemble applications, servers, and tools.

Installation

All the _.py components are hosted on PyPi and GitHub.

$ pip install underscorepy
$ pip install underscorepy_web
$ pip install underscorepy_storage
$ pip install underscorepy_pb
